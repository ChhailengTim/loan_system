<h2 align="center">Setup Project</h2>

## Required Software

- Wamp, Xampp server or another server
- Composer
- Node Js
- Git
## Api

Run Command as following

- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan serve

Config Database and Email In .env file

## Front

Run Command as following

- npm install
- npm run watch
