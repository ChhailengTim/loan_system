<p align="center">Setup Project</p>

## Required Software

- [wampserver].
- [Composer].
- [Node Js].
- [Git].
## Api

Run Command as following

- [composer install].
- [cp .env.example .env].
- [php artisan key:generate].
- [php artisan migrate].
- [php artisan db:seed].
- [php artisan serve].

## Front

Run Command as following

- [npm install].
- [npm run watch].
